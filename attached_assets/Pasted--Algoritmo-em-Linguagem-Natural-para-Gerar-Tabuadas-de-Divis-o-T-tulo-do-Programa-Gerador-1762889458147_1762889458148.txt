## Algoritmo em Linguagem Natural para Gerar Tabuadas de Divisão

### Título do Programa
"Gerador de Tabuadas de Divisão Completas"

### Descrição para o Replit
Este programa deve gerar e exibir as tabuadas de divisão de 2 a 10, mostrando divisões que resultam em números inteiros de 1 a 10.

### Algoritmo Passo a Passo

**INÍCIO DO PROGRAMA**

1. **Exibir cabeçalho principal**
   - Imprimir "TABUADA DE DIVISÃO" centralizado
   - Pular duas linhas

2. **Criar loop principal para cada divisor**
   - PARA cada divisor de 2 ATÉ 10, FAÇA:

3. **Exibir cabeçalho de cada tabuada**
   - Imprimir uma linha com "=" (40 caracteres)
   - Imprimir "÷ [divisor] (Dividindo por [divisor])"
   - Imprimir uma linha com "=" (40 caracteres)

4. **Gerar cada linha da tabuada**
   - PARA cada resultado de 1 ATÉ 10, FAÇA:
     - Calcular o dividendo: dividendo = divisor × resultado
     - Imprimir "[dividendo] ÷ [divisor] = [resultado]"
     - Alinhar os números para ficarem organizados em colunas

5. **Adicionar espaçamento**
   - Após cada tabuada completa, pular uma linha em branco

6. **Opções extras para o programa**
   - Perguntar ao usuário se deseja:
     a) Ver todas as tabuadas de uma vez
     b) Ver uma tabuada específica
     c) Salvar as tabuadas em um arquivo texto

7. **Para versão interativa**
   - SE o usuário escolher tabuada específica:
     - Pedir para digitar o número do divisor (2 a 10)
     - Validar se o número está entre 2 e 10
     - Mostrar apenas a tabuada solicitada
     - Perguntar se deseja ver outra

8. **Formatação da saída**
   - Usar espaços para alinhar os sinais de igual
   - Garantir que números de dois dígitos fiquem alinhados
   - Exemplo de formato: "24 ÷ 6 = 4" (com espaços consistentes)

9. **Função para salvar em arquivo (opcional)**
   - SE o usuário escolher salvar:
     - Criar arquivo "tabuada_divisao.txt"
     - Escrever todas as tabuadas no arquivo
     - Informar "Arquivo salvo com sucesso!"

**FIM DO PROGRAMA**

### Dicas de Implementação para o Replit

1. Use dois loops aninhados (um para o divisor, outro para o multiplicador)
2. Para alinhar números, use formatação de string com largura fixa
3. Considere adicionar cores no terminal para destacar os cabeçalhos
4. Teste primeiro com divisores 2 e 3 antes de fazer todos
5. Adicione comentários explicando cada parte do código
